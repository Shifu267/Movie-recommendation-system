## 📌 Overview

**Cinemascope** is a sophisticated content-based movie recommendation system that leverages machine learning algorithms to provide personalized movie suggestions. The system analyzes movie metadata, including genres, keywords, cast, and crew, to find similar movies based on cosine similarity and K-Means clustering.

### 🎯 What Makes Cinemascope Special?

- **🎨 Stunning Visual Experience** - Black & gold cinematic theme with animated backgrounds
- **🧠 Intelligent Recommendations** - Content-based filtering with 5000+ movies
- **⚡ Real-time Processing** - Instant recommendations with high accuracy
- **📊 Advanced Analytics** - Cluster analysis and similarity visualizations
- **🎭 Movie Comparison** - Compare any two movies side-by-side

## ✨ Features

### Core Features

| Feature | Description |
|---------|-------------|
| 🎯 **Movie Recommendations** | Get top-N similar movies based on content similarity |
| ⚖️ **Movie Comparison** | Compare two movies and see their similarity score |
| 🔭 **Cluster Explorer** | Explore movies grouped by K-Means clusters |
| 📊 **Visual Analytics** | Interactive charts showing genre distribution and cluster sizes |
| 🔍 **Smart Search** | Search and filter movies across the database |
| 🎨 **Cinematic UI** | Animated backgrounds, glowing effects, and smooth transitions |

### Technical Features

- ✅ Content-based filtering with CountVectorizer
- ✅ Cosine similarity for movie matching
- ✅ K-Means clustering (optimal clusters via elbow method)
- ✅ L2 normalization for feature scaling
- ✅ PCA for dimensionality reduction
- ✅ Association rule mining for genre relationships
- ✅ Star schema for data warehousing
